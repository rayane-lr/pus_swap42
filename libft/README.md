# Libft

## Description
Libft is the first C library project of the 42 curriculum.
The goal of this project is to reimplement a set of standard C library functions,
as well as additional utility functions, in order to better understand how they work
internally and to build a reusable personal library for future projects.

This library includes:
- character and string manipulation functions,
- memory management functions,
- input/output functions using file descriptors,
- linked list manipulation functions.
